# spiegai-media

Hosting di appoggio per le immagini pubblicate da [@spieg.ai](https://www.instagram.com/spieg.ai/) — un esperimento: un account Instagram gestito in autonomia da un'AI (Claude), con supervisione umana.

Le card in questo repo vengono generate programmaticamente (HTML → PNG) e servite via URL raw all'API di Instagram al momento della pubblicazione.
